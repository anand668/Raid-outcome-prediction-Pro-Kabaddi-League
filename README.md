# Raid-outcome-prediction-Pro-Kabaddi-League
This repository has the implementation of the study:- Raid outcome prediction in Pro Kabaddi League. which has objectives as below:-
1.Raid outcome classification (Empty/Successful/Unsuccessful)
2.Expected Raid Points
3.Live win probability
All these objectives are implemented with ML/DL models on Google Colab Notebook.
All these implemented on Pro Kabaddi Leagues season(6-11) Matches data.
The data is collected from the official cite of PRO KABADDI LEAGUE ( https://www.prokabaddi.com/schedule-fixtures-results )
